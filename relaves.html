PRESA DE RELAVES

Es el sitio donde se almacena los residuos mineros. Para construirlos se sigue el siguiente procedimiento:

EXVACACIÓN

Es el movimiento de tierras para iniciar la edificación.

CIMENTACIÓN

Viene a ser la base sobre la cual estara apoyada la presa.

CONSTRUCCION DE TERRAPLEN

Viene a ser la construccion de una base encima de la cimentación para tener una base poco deformable.

COMPACTACIÓN DEL TERRAPLEN

Se hace la compactación para tener una base sólida y evitar posibles fugas

CONSTRUCCION DE MUROS DE CONTENCIÓN

Su función es evitar el derrame de residuos mineros.

PRESA DE RELAVE TERMINADA

Una vez terminada, se procede a trasladar los residuos mineros.
